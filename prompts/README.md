# Prompts Directory

A curated collection of prompting patterns for senior software development with CLI-based AI coding assistants.

## Available Prompts

- **[general.md](./general.md)** - Overview of the prompts library, usage guide, and credits
- **[autonomous-agents.md](./autonomous-agents.md)** - Self-directed coding agent patterns
- **[tool-usage-patterns.md](./tool-usage-patterns.md)** - Guidelines for effective tool invocation
- **[planning-and-tracking.md](./planning-and-tracking.md)** - Task breakdown and progress tracking
- **[design-constraints.md](./design-constraints.md)** - Enforcing code standards and design systems
- **[metaprompting.md](./metaprompting.md)** - Diagnosing and optimizing prompt effectiveness

## Quick Start

1. Browse [general.md](./general.md) for an overview
2. Pick a category that matches your needs
3. Copy and adapt patterns to your context

Each file contains concrete examples ready to use with Claude Code, Codex CLI, and similar AI assistants.
